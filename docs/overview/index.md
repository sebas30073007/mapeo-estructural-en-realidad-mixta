# Visión General

Este proyecto desarrolla un sistema robótico autónomo capaz de mapear espacios interiores y representar esa información en tiempo real dentro de una aplicación de realidad mixta. El operador controla el robot de forma remota usando lentes de realidad mixta, visualiza una reconstrucción digital del entorno y puede analizar métricas capturadas durante el recorrido.

La aplicación en desarrollo es la generación de **mapas de calor de intensidad de señal WiFi**: el robot recorre un espacio como lo haría una Roomba, midiendo la potencia de red en cada punto del área y construyendo una representación visual de la cobertura. Con esa información se pueden tomar decisiones informadas sobre la ubicación óptima del módem o identificar estructuras que degraden la señal.
