# Objetivo del Proyecto

El objetivo del proyecto es desarrollar un sistema robótico capaz de mapear un entorno físico y representar su estructura dentro de una aplicación de realidad mixta.

El sistema busca facilitar la exploración remota de espacios interiores mediante un robot omnidireccional, permitiendo que el operador observe el entorno desde el video del robot, controle su movimiento y visualice una reconstrucción digital de las paredes detectadas.
