# Aplicación en Realidad Mixta

La interfaz de usuario del sistema es una aplicación de realidad mixta desarrollada en **Unity 2022.3.58f1** para los lentes **Meta Quest**, usando el **Meta XR All-in-One SDK v83**. Permite al operador controlar el robot de forma remota, visualizar el video en tiempo real y observar la reconstrucción digital del espacio escaneado, todo desde el visor sin necesidad de estar físicamente presente en el área de trabajo.

La aplicación se comunica con el robot exclusivamente mediante **UDP** a través de la red virtual de Tailscale.
